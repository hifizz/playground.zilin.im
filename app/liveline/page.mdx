This is a tribute / re-skin of [benji.org/liveline](https://benji.org/liveline). Same library, same demos, our own typography sandbox — built to study how a single column of muted text plus a few embedded charts feels so much more alive than yet another marketing landing page.

The library is **liveline** by [@benjitaylor](https://github.com/benjitaylor/liveline). It does *one thing* — draws a line that moves smoothly when a number changes — and everything else (momentum, candles, orderbook, multi-series, degen particles) is opt-in.

## Getting started

Pass a growing array of `{ time, value }` points and the latest value. Liveline interpolates between updates so even sparse data renders smooth at 60fps.

```jsx
import { Liveline } from 'liveline'

function Chart({ data, value }) {
  return (
    <div style={{ height: 200 }}>
      <Liveline data={data} value={value} />
    </div>
  )
}
```

## Multi-series

Pass `series` instead of `data` to draw multiple overlapping lines that share one axis. Toggle chips appear when there are 2+ series — clicking hides that line with a smooth fade and the Y-axis re-fits.
